# DotVault documentation

DotVault is an open-source vault for `.env` files: **quick share** (ephemeral, zero-knowledge links) and a **cloud vault** (authenticated projects with encrypted storage).

Use the sidebar to browse guides for setup, the CLI, browser extension, security controls, and integrations.
